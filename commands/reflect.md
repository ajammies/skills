# Reflect on Session

Analyze the session using the `reflect` skill workflow.

## Usage

```
/reflect
```

## Behavior

1. **Invoke `reflect` skill** to analyze:
   - Full conversation history
   - Git commits made during session
   - User corrections and preferences
   - Friction points and inefficiencies

2. **Output recommendations** for workflow improvements

## Examples

```
/reflect                          # Analyze current session
```
